Mention control

A multiple select control for selecting users with custom dropdown rendered. It can return multiple types of values. Requires external list of data in jsonforms config

String value

You can choose in uischema options the key Person object which you want to select. Default email is used

Schema

Ui Schema

Data

Whole object

When you pass all as return key whole object will be used as a value of input
