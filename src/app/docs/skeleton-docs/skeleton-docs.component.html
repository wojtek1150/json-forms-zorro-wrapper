Loading skeleton

This is a simple example of how to use the loading skeleton. Simply add loadingSkeleton property with true/false (false by default) to jsonforms component. This will show a skeleton of the form.

Theme options

Variable Description Default --jsfw-skeleton-control-gradient-start Start color of the gradient #5462fb --jsfw-skeleton-control-gradient-end End color of the gradient #549ffb --jsfw-skeleton-control-background Background color of the control #fafafa --jsfw-skeleton-control-height Height of the control 32px
